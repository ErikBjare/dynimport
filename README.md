dynimport
=========

*Import Python packages from anywhere!*

dynimport installs & imports a Python package for you, from anywhere.


### Examples

```python
from dynimport import dimport

ttt = dimport("takethetime@pypi")

with ttt("Time this task"):
    print("Hello world!")
```


### Use-cases

 - Notebooks
 - Scripts
